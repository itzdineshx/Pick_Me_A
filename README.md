## ğŸš€ Quick Start

### Prerequisites
- Modern web browser (Chrome 90+, Firefox 88+, Safari 14+)
- No build tools required - pure HTML/CSS/JS

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/pick-me-a.git
   cd pick-me-a
   ```

2. **Open in browser**
   ```bash
   # Using Python (if available)
   python -m http.server 8000
   
   # Using Node.js (if available)
   npx serve .
   
   # Or simply open index.html in your browser
   ```

3. **View the site**
   - Navigate to `http://localhost:8000` (if using server)
   - Or open `index.html` directly in your browser

### Development

1. **Local Development**
   ```bash
   # Install live server (optional)
   npm install -g live-server
   
   # Run with hot reload
   live-server --port=3000
   ```

2. **Code Structure**
   - Edit `index.html` for content changes
   - Modify `css/style.css` for styling updates  
   - Update `js/main.js` for functionality changes

## ğŸ¨ Customization

### Changing Colors
```css
:root {
  --anime-color: #dc2626;    /* AniePick red */
  --movie-color: #fbbf24;    /* CinePick gold */
  --music-color: #10b981;    /* MeloPick green */
  --background: #000000;     /* Main background */
  --text-primary: #ffffff;   /* Primary text */
}
```

### Adding New Platforms
1. Create new card in `index.html`
2. Add corresponding styles in `css/style.css`
3. Update particle system in `js/particles.js`
4. Add hover animations in `js/animations.js`

### Modifying Animations
- **Duration**: Change CSS `transition` values
- **Easing**: Modify `cubic-bezier()` functions
- **Effects**: Update transform properties
- **Particles**: Adjust parameters in `particles.js`
